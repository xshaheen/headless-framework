# Headless Shop Template

Installs the `headless-shop` .NET template. The generated app is an opinionated Headless Framework capability tour for modular APIs, tenant posture, messaging boundaries, and agent-readable extension rules.

```bash
dotnet new install Headless.Templates.HeadlessShop
dotnet new headless-shop -n TrailStore
```

The template intentionally ships one `tour` shape. Generated output includes `AGENTS.md`, architecture notes, validation guidance, and an add-command recipe that are validated by the source repo template gate.
