# Code Review Summary

PR #194 adds per-consumer-group circuit breaker (Closed→Open→HalfOpen) with adaptive retry backpressure across 8 messaging transports. The core architecture is sound — lock-free fast paths on IsOpen, generation-based timer guards, TCS-based resume tracking, and adaptive CAS loops. The custom circuit breaker (vs. Polly) is correctly motivated. Three critical issues block merge.

## Key Findings

### P1 Critical (Blocks Merge)
1. **Escalation counter bug** — `_TransitionToClosed` credits `SuccessfulCyclesAfterClose` for non-transient-failure closes, giving false recovery credit
2. **Log injection** — `ex.Message` and raw transport name/group flow unsanitized into structured log params via exception wrapping
3. **TOCTOU race** — Transport `PauseAsync`/`ResumeAsync` check `IsPaused` then act non-atomically; concurrent callers can double-execute transport-level operations

### P2 Important
- Pre-cache OTel tag strings (hot path allocation)
- `RemoveGroup` sync timer dispose + `Dispose()` ResumeTask gap
- README `ResetAsync` compile error (includes non-existent CancellationToken param)
- `ConsumerCircuitBreakerOptions` missing FluentValidation validator
- `ForceOpenAsync`/`ResetAsync` lack authorization guidance
- Volatile non-atomic increments should use `Interlocked`
- `RegisterGroupCallbacks` silent callback overwrite
- Timer disposal inside lock in `_TransitionToOpen`
- No escalation reset API
- `ICircuitBreakerMonitor` naming misleading (3 reviewers flagged independently)

### P3 Nice-to-Have
- `TaskCompletionSource<bool>` → non-generic
- Dead `tripped` variable + double `_GetOpenDuration`
- Collapse `Register`/`RegisterOrUpdate`
- `_capWarningLogged` volatile → Interlocked
- `GetAllStates` return type
- LogSanitizer truncation indicator
- XML docs "Gets" → "Gets or sets"
- Style: `s_noOpState` prefix, LogSanitizer indentation
- `ForceOpenAsync` missing OTel + LogSanitizer inside lock
- Plan: EventCounterSource counters skipped, integration test gap
- `_ReopenAfterResumeFailureAsync` reads EscalationLevel outside lock
